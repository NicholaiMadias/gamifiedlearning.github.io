# Audio Assets

Place the following audio files in this directory:

| File              | Format | Description                              |
|-------------------|--------|------------------------------------------|
| `match.wav`       | WAV    | Played on a successful gem match         |
| `combo.wav`       | WAV    | Played when a combo is triggered         |
| `revelation.mp3`  | MP3    | Played when the Revelation Engine fires  |

All files should be short (< 3 seconds) and optimised for web playback.
