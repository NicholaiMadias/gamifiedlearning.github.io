# gamifiedlearning.github.io
Mobile friendly educational gaming, Bible study, admin console, sandbox app and interactive simulations.

## Games

### The Mystery of the Seven Stars

A star-matching puzzle game. Click stars to match them and decrypt the biometrics.

**Hosted path:** `/games/seven-stars/`

**Local dev:**
```bash
# No build step required — open directly in a browser
open games/seven-stars/index.html
# or serve with any static file server, e.g.:
npx serve .
# then visit http://localhost:3000/games/seven-stars/
```

**Files:**
- `games/seven-stars/index.html` — game entry point
- `games/seven-stars/script.js` — game logic
- `games/seven-stars/style.css` — styles and animations
- `games/seven-stars/assets/` — star crystals, starfield background, supernova image
